### 🚂 [Exploratory Data Analysis — Indian Railways](https://github.com/vidhan143)
**EDA & Data Visualization** • *Python, Pandas, Matplotlib, Seaborn*

Deep-dived into India's massive railway dataset to uncover operational patterns, delay trends, and route efficiency insights.

- **Data Cleaning**: Handled missing values, outliers, and formatting inconsistencies
- **Visualizations**: Bar charts, heatmaps, and line graphs for delay & passenger traffic trends
- **Insights**: Identified peak congestion routes and performance bottlenecks
- **Tools**: Jupyter Notebook for reproducible analysis

<details>
<summary>🔍 Analysis Highlights</summary>

**EDA Pipeline:**
```python
# My analysis workflow
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 1. Load & inspect data
df = pd.read_csv("indian_railways.csv")
df.info(), df.describe(), df.isnull().sum()

# 2. Clean & preprocess
df.dropna(subset=["delay_minutes"], inplace=True)
df["route"] = df["source"] + " → " + df["destination"]

# 3. Visualize patterns
sns.heatmap(df.corr(), annot=True)         # Correlation matrix
df.groupby("zone")["delay"].mean().plot()  # Delay by zone
sns.barplot(x="train_type", y="occupancy") # Traffic by type
```

**Key Findings:**
- Seasonal delay spikes tied to monsoon months
- Northern zones showed highest average delays
- Express trains had 2x the on-time rate vs. passenger trains

</details>
